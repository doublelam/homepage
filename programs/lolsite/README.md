# Web Front-End Starter Kit

## 一、简介

一款轻量级的 Web 前端开发框架，提供简单、快速的 Web 前端开发体验。

## 二、特性

* 仅针对Web前端开发，兼容IE8+。
* 遵循 MIT 协议，无论公司还是个人，都可以免费、自由使用。
* 提供常用的样式及交互组件，能帮助您轻松构建 Web 应用。
* 采用 LESS 技术、CMD 规范、jQuery 插件开发模式，扩展方便。

* [Squirrel 3](http://hisquirrel.com)
